# KaspiBoard Mobile (Flutter)

Простое мобильное приложение, которое получает ленту объявлений с Django API

## Что делает

- Запрашивает `GET /api/ads/`
- Показывает ленту карточек объявлений
- Отмечает VIP-объявления бейджем и рамкой
- Поддерживает pull-to-refresh

## Запуск

1. Убедитесь, что Django-сервер запущен:

```bash
python manage.py runserver 0.0.0.0:8000
```

2. Перейдите в папку `mobile_flutter` и установите зависимости:

```bash
flutter pub get
```

3. Запустите приложение:

```bash
flutter run
```

В файле `lib/main.dart`:

- для Android-эмулятора используйте `http://10.0.2.2:8000`
- для iOS simulator обычно можно `http://localhost:8000`
- для реального телефона укажите IP вашего компьютера в локальной сети, например:
  `http://192.168.1.15:8000`

Меняется в константе:

`AdsApi.baseUrl`
